# Product Overview

## 产品定位

Agent Flow Lite 是一个轻量级 AI 智能体编排平台，面向需要快速搭建工作流、对话、RAG 知识库和 Skill 体系的团队。

## 核心价值

- 用可视化工作流降低智能体编排门槛
- 用 RAG 和 Skill 系统提升知识复用与任务自动化能力
- 用统一的前后端架构支持从 Demo 到可持续迭代的工程实践

## 主要使用者

- 产品或运营：通过工作流和对话快速验证场景
- 开发者：扩展节点、Skill、RAG 和 API 能力
- 管理员：管理用户、会话和知识库

## 关键业务对象

- Workflow：可视化编排后的执行图
- Knowledge Base：上传文档后建立的检索索引
- Skill：可复用的 Prompt / 工具能力单元
- Chat Session：对话历史与 SSE 响应流
- User / Admin：认证与权限边界

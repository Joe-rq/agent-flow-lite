# Python 编程语言

## 简介

Python 是一种高级、解释型、通用的编程语言。由 Guido van Rossum 于1991年创建，以简洁、易读的语法著称。Python 的设计理念强调代码的可读性和简洁的语法，使用缩进来表示代码块。

Python 是目前世界上最流行的编程语言之一，在 TIOBE 指数和 GitHub 使用量排名中常年位居前三。

## 主要特性

### 1. 简洁优雅的语法

Python 使用缩进来定义代码块，语法接近自然语言：

```python
# Python 的 Hello World
print("Hello, World!")

# 简单的循环
for i in range(5):
    print(i)
```

### 2. 动态类型系统

Python 是动态类型语言，无需声明变量类型：

```python
name = "Python"  # 字符串
version = 3.12   # 数字
is_great = True  # 布尔值
```

### 3. 丰富的标准库

Python 自带"电池"，标准库包含：
- 文件 I/O 操作
- 网络编程（urllib、http）
- 数据库接口（sqlite3）
- 数据处理（json、csv、re）
- 多线程/多进程支持

### 4. 跨平台

Python 可以在 Windows、macOS、Linux 等操作系统上运行，代码具有良好的可移植性。

## 应用领域

### Web 开发

Python 拥有多个流行的 Web 框架：
- **Django**：功能齐全的 MVC 框架
- **Flask**：轻量级微框架
- **FastAPI**：现代高性能异步框架

### 数据科学与机器学习

Python 是数据科学领域的主导语言：
- **NumPy**：数值计算库
- **Pandas**：数据处理和分析
- **Matplotlib/Seaborn**：数据可视化
- **Scikit-learn**：机器学习库
- **TensorFlow/PyTorch**：深度学习框架

### 自动化与脚本

Python 广泛用于自动化任务：
- 系统管理脚本
- 文件处理自动化
- 网络爬虫（Scrapy、BeautifulSoup）
- 测试自动化

### 人工智能

Python 是 AI 开发的首选语言：
- OpenAI 的 GPT 模型主要使用 Python
- 自然语言处理（NLTK、spaCy、Transformers）
- 计算机视觉（OpenCV、PIL）

## Python 版本

当前主要版本：
- **Python 3.12**（2023年10月发布）
- **Python 3.11**（性能大幅提升）
- **Python 3.10**（引入模式匹配）

## 代码示例

### 基础语法

```python
# 变量和数据类型
count = 100          # 整数
price = 19.99        # 浮点数
name = "Python"      # 字符串
items = [1, 2, 3]    # 列表
data = {"a": 1}      # 字典

# 条件语句
if count > 50:
    print("Count is large")
elif count > 25:
    print("Count is medium")
else:
    print("Count is small")

# 循环
for item in items:
    print(item)

# 函数定义
def greet(name):
    return f"Hello, {name}!"

# 类定义
class Person:
    def __init__(self, name):
        self.name = name
    
    def say_hello(self):
        return f"Hi, I'm {self.name}"
```

### 文件操作

```python
# 读取文件
with open('file.txt', 'r') as f:
    content = f.read()

# 写入文件
with open('output.txt', 'w') as f:
    f.write("Hello, World!")
```

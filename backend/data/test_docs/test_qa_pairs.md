## 测试评分标准

### 准确性评分

- **完全正确（5分）**：答案与预期完全一致
- **基本正确（4分）**：答案包含核心信息，略有偏差
- **部分正确（3分）**：答案包含部分正确信息
- **错误（1分）**：答案明显错误
- **无法回答（0分）**：系统无法提供答案

### 测试通过标准

- 总分 >= 100 分（满分 120 分）：优秀
- 总分 >= 90 分：良好
- 总分 >= 75 分：合格
- 总分 < 75 分：需要优化
